The Entropic Archive

Interactive Black Hole Simulation

Project Overview

The Entropic Archive is an immersive web experience that visualizes black hole physics through interactive data decay. Users submit information that gradually distortsâ€”simulating spaghettification, Hawking radiation, and the information paradox. The project transforms abstract astrophysics concepts into tangible UI behaviors.

Key Features

  • Event Horizon Simulation: User inputs collapse into the archive with gravitational animations, disappearing beyond the "point of no return".
  • Time-Based Entropy: Implemented progressive data corruption where information decays into glitches and noise over time.
  • Hawking Radiation Visualization: Created particle systems that leak "data remnants" using Canvas API.
  • Spaghettification Effect: CSS transforms stretch UI elements vertically as they age, mimicking tidal forces.
  • No-Escape Mechanism: Disabled back/refresh actions to enhance immersion in the black hole metaphor.

Technologies Used

Core Framework
Django for backend data processing
Frontend
HTML5/CSS3 animations, Canvas API for particle effects
Physics Simulation
Custom time-based entropy algorithms in Python
UI Effects
CSS transforms, keyframe animations, and JavaScript event handling
Concept Inspiration
Hawking radiation, information paradox, and general relativity principles

Project Impact

This artistic interpretation of astrophysics demonstrates how complex scientific concepts can be made accessible through interactive design. The project pushes boundaries in:

  • Experimental UI: Challenges conventional web interactions by implementing "destructive" interfaces
  • Science Communication: Visualizes abstract physics theories in an engaging format
  • Generative Art: Creates unique decay patterns for each user submission
