Project Management System

Java/Kotlin Application

Project Overview

The aim of the project was to develop a project management system that allows users to create, edit, and manage tasks within a project. The system aimed to provide a comprehensive tool for task management, incorporating features like task assignment, adjacency matrix representation, and domain graph visualisation.

Deliverables

  • Domain Knowledge and Graph: Created domain and entity classes in Kotlin to represent tasks as graphs, supporting the Java-based GUI application. This structure allowed users to visualise project dependencies and tasks in a structured manner.
  • Adjacency Matrix Implementation: Developed an adjacency matrix in Kotlin, enabling users to view task dependencies within a project. This matrix was integrated into the GUI, offering a clear representation of the project's structure.
  • Desktop Application with GUI: Developed a desktop application using Java, providing users with the ability to create, edit, and delete projects and their tasks through a graphical user interface.

Technologies Used

Programming Language
Java for the desktop application with GUI
Programming Language
Kotlin for domain classes and graph representation
Data Structure
Adjacency matrix for task dependency visualization
